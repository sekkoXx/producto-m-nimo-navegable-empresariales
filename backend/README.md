# Backend — Instrucciones

Crear entorno virtual:

```bash
python -m venv venv
```

Activar entorno (Windows):

```bash
venv\Scripts\activate
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

Crear base de datos (genera `database.db` en el directorio `backend/`):

```bash
python init_db.py
```

Ejecutar servidor:

```bash
python app.py
```
